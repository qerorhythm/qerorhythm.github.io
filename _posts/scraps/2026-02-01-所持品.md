## オーディオ

### Soundcore P40i / ZE500

| Brand | ANKER / final |
| Model | Soundcore P40i / ZE500 for ASMR |
| Color | Black / Cream |
| Date | 2024-12 / 2025-7 |
| Link | https://www.ankerjapan.com/products/ https://final-inc.com/products/ |

{% include figure image_path="/assets/images/posts/ramblings/gear/1769092983034.jpg" alt="" caption="" class="half" %}

イヤホンたち。  
P40iは普段使い、ZE500は寝る時用です。  
ワイヤレスイヤホンは消耗品だと思っているのでまた変えますが、ANKERのものはコスパがいいので、音の違いがよく分からない自分に丁度いいです。  
昔からテレビを聞きながら寝ると寝付きがいいので、寝る時も雑談動画か雨音流しながら寝ています。  
ZE500は小さくて睡眠イヤホンに向いていますが、朝起きたら絶対行方不明になっているのでそれだけ面倒です。  
一時は左右が繋がっているタイプのワイヤレスを使っていましたが、やはり有線はコード接触のガサガサ音が気になってしまいました。
